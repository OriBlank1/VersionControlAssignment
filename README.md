Version control is a system that tracks and manages changes made to a file, which allows users to see all versions of a file. Version control is so important because it actively backs up and protects your work and data so if something bad were to happen to
your computer, your work would be saved through a cloud.

Git is the process that allows your work to be uploaded to a clodu. GitHub itself is your own individual cloud where you can safely store and secure your data and work.
